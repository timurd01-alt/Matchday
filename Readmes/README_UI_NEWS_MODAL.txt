UI-only patch.

Copy only index.html into your Matchday folder and replace the old one.

This patch does not touch fetch_data.py, config_keys.py, app.py, data.json, API keys, model picks, odds, or launcher files.

Changes:
- News source detection is more robust and the All view is interleaved by source.
- Right-side insight news uses a balanced source rotation.
- Fixture cards no longer expand downward.
- Clicking a fixture opens a larger modal with model, odds, stats, and lineups.
- Escape key or the X button closes the modal.
